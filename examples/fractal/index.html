2D Fractal Rendering - Julia Set

This app renders 2D Julia set fractals for a variety of input parameters using a WebGL fragment shader. Written by Amanda Ghassaei.

Instructions: Zoom/scroll and pan to explore each fractal's structure. Adjust "C Real" and "C Imaginary" to generate different Julia sets, and increase "Max Iters" to see more detail in the brightest regions.

This page is part of a series of examples using gpu-io, a web-based GPU-accelerated computing library for physics simulations, cellular automata, and other mathematical calculations. Find the source code on Github.
