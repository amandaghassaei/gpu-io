Physarum Transport Network

This system simulates the network effects of a multicellular slime mold (Physarum polycephalum), based on work by Sage Jenson and a 2010 paper by Jeff Jones: "Characteristics of pattern formation and evolution in approximations of Physarum transport networks." In this simulation, particle "agents" interact with an underlying field of chemical attractants to form complex, emergent behaviors. All simulation steps are computed on the GPU via a series of WebGL fragment shaders. Written by Amanda Ghassaei.

This page is part of a series of examples using gpu-io, a GPU-accelerated computing library for physics simulations and cellular automata. Find the source code on Github.
